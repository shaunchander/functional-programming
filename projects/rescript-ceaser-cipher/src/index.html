ReScript — Ceaser's Cipher

Ceaser's cipher is the simplest form of cryptography. How it works is you pass in a message that you want to encrypt (or decrypt) as well as some shift value. Letters are then converted to their numerical position in the alphabet plus or minus the shift value and new cipher string is returned. This Ceaser's Cipher app was built on top of ReScript as an intro to the programming language. Checkout the source code!

Encrypt some messages and send them to your friends or decrypt messages you've recieved. Make sure that you send the shift key you used for encryption or have a shift key from someone for the message you are trying to encrypt/decrypt.
